Acerca de esta prueba

Este proyecto práctico tiene como objetivo evaluar su capacidad para implementar un servidor HTTP usando Node.js, interactuar con archivos estáticos y gestionar una base de datos SQLite.

Se espera que los estudiantes:

  • Configuren un entorno de desarrollo Node.js (10%).
  • Creen y configuren un servidor HTTP básico (30%).
  • Sirvan archivos HTML y CSS de manera efectiva (30%).
  • Establezcan una conexión funcional con una base de datos SQLite (20%).
  • Manejen el cierre seguro del servidor y la base de datos (10%).
